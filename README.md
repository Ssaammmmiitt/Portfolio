# Sammit Poudyal — Portfolio

Personal portfolio site for **Sammit Poudyal**, a fullstack developer and AI/ML enthusiast. The site presents work, services, tech stack, and contact information in a single-page layout with a strong visual identity.

## Overview

The portfolio is built with **React** and **Vite**, styled with **Tailwind CSS**, and animated with **Framer Motion**. It includes a dark/light theme, a WebGL terminal-style background (`FaultyTerminal`), a 3D hero scene (Spline), scrolling tech logos, and interactive project showcases.

## Sections

- **Hero** — Introduction, role, and social links with a 3D accent
- **About** — Background and focus areas
- **Services** — What I offer as a developer
- **Work** — Selected projects
- **Tech Stack** — Animated logo marquee of tools and technologies
- **Contact** — Email, GitHub, and LinkedIn

## Tech Stack

React · Vite · Tailwind CSS · Framer Motion · GSAP · Lenis · Spline · OGL · React Icons
